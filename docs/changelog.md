# Changelog

A friendly summary of what's changed in Banwatch. The newest version is at the
top.

> This page is maintained by hand. When you ship a new build, add a short,
> plain-language entry here so server owners can see what's new.

## 3.4 — Warning system

- Added a full **warning system**. Give members warnings with `/warnings warn`,
  review them with `/warnings manage`, and set automatic timeouts, kicks, or
  bans once someone reaches a threshold with `/warnings punishments`. See
  [Warnings](warnings.md).

## 3.2

- General improvements and fixes across bans, appeals, and evidence.

## 3.0 — Evidence & verification

- Introduced the **evidence management system** for attaching proof to bans. See
  [Evidence](evidence.md).
- Added **verified bans** — bans reviewed and confirmed by the Banwatch team.
- New sharing choices when you ban someone: *share*, *share with proof*,
  *silent*, and *hide*. See [Bans](bans.md).

## Earlier

- The core of Banwatch: ban monitoring, cross-network lookups, join-time
  warnings, real-time ban updates, and invites back to a banned user's origin
  server.
